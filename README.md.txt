The program is to automatcally gather url and English and Chinese text strings
from json files for the proofread work for course translation.

How to run:
- Download proofread.py
- Get json files for translation
- run "python proofread.py [filename_1] [filename_2]", while [filename_1] and
  [filename_2] are names of json files
- The result can be found in the output .md file
